Introduction

This privacy policy explains how the Productivity Shield browser extension handles user data and ensures privacy protection.

Data Collection and Usage

This Chrome extension ("Productivity Shield") does not collect, transmit, or share any personally identifiable information. The extension only stores user preferences (e.g., which platforms to block) locally using Chrome's storage.sync API. This data is used solely to provide the core functionality of the extension — blocking YouTube Shorts, Instagram Reels, Explore, and related content — and is never shared with external services.

Permissions

The extension requires access to youtube.com and instagram.com in order to inject CSS rules that hide short-form content and to detect navigation to blocked pages. No data from these sites is read, collected, or transmitted.

Third-Party Sharing

We do not sell, trade, or transfer user data to any third parties.
